# Memory Organization in 8085

## What is Memory?

Memory is used to store instructions and data required by the microprocessor during program execution.

---

## Memory Size

The 8085 has a **16-bit address bus**, allowing it to access **64 KB (65,536 bytes)** of memory.

**Address Range:**

```
0000H to FFFFH
```

---

## Types of Memory

### 1. RAM (Random Access Memory)

- Read and write memory.
- Stores temporary data.
- Data is lost when power is turned off.

---

### 2. ROM (Read Only Memory)

- Stores permanent data and programs.
- Data is retained even when power is turned off.

---

## Memory Map

```
FFFFH  ─────────────────────
        Memory Space
0000H  ─────────────────────
```

---

## Memory Mapped I/O

In Memory Mapped I/O, input/output devices are assigned memory addresses.

**Features:**
- Uses the 16-bit address bus.
- Can access all memory instructions.
- Reduces available memory space.

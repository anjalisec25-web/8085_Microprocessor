# I/O Organization in 8085

## What is I/O Organization?

I/O Organization is the method used by the microprocessor to communicate with input and output devices.

---

## Types of I/O Organization

### 1. Memory Mapped I/O

- Devices are assigned memory addresses.
- Uses a 16-bit address.
- Supports all memory instructions.

---

### 2. I/O Mapped I/O

- Devices are assigned separate I/O port addresses.
- Uses an 8-bit port address.
- Accessed using **IN** and **OUT** instructions.

---

## Difference Between Memory Mapped I/O and I/O Mapped I/O

| Memory Mapped I/O | I/O Mapped I/O |
|-------------------|----------------|
| Uses 16-bit addresses. | Uses 8-bit port addresses. |
| Shares memory address space. | Uses a separate I/O address space. |
| Supports all memory instructions. | Uses only `IN` and `OUT` instructions. |
| Reduces available memory. | Does not reduce memory space. |

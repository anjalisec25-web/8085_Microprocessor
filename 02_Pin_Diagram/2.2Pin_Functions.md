# 8085 Pin Functions

## 1. Address Bus (A8–A15)

- Higher-order 8-bit address lines.
- Used to send memory addresses.
- Unidirectional.

---

## 2. Address/Data Bus (AD0–AD7)

- During first cycle acts as address bus.
- Later acts as data bus.
- Bidirectional.
- These pins are multiplexed.

---

## 3. ALE (Address Latch Enable)

- Separates the address and data from the multiplexed lines.
- Used to latch the lower-order address.

---

## 4. RD̅ (Read)

- Indicates that the processor is reading data from memory or an I/O device.
- Active Low(i.e The read operation happens when RD̅ is low-0).

---

## 5. WR̅ (Write)

- Indicates that the processor is writing data to memory or an I/O device.
- Active Low(i.e The write operation happens when WD̅ is low-0).

---

## 6. IO/M̅

- Distinguishes between memory and I/O operations.
- High → I/O operation.
- Low → Memory operation.

---
## 7. S0 and S1 Pins 

- S0 and S1 are status output pins.
- They indicate the current operation of the 8085.
- They are used along with the IO/M̅ signal to identify the machine cycle.

| **S1** | **S0** | **Operation** |
|:------:|:------:|---------------|
| 0 | 0 | Halt (Processor stopped) |
| 0 | 1 | Write data |
| 1 | 0 | Read data |
| 1 | 1 | Fetch instruction (Opcode Fetch) |

---

## 8. READY

- Indicates whether the connected device is ready.
- If LOW, the processor waits until the device becomes ready.

---

## 9. HOLD

- Requests control of the system buses for DMA operations.

---

## 10. HLDA (Hold Acknowledge)

- Acknowledges the HOLD request.(i.e,CPU gives Control).
- Indicates that the buses have been released.

---

## 11. RESET IN̅

- Resets the microprocessor.
- Clears the Program Counter.
- Execution starts from address `0000H`.

---

## 12. RESET OUT

- Sends a reset signal to external devices connected to the system.

---

## 13. X1 and X2

- Used to connect the crystal oscillator.
- Generate the system clock.

---

## 14. CLK OUT

- Provides the clock signal to external devices.

---

## 15. TRAP

- Highest-priority interrupt.
- Non-maskable interrupt.(i.e,CPU must handle it immediately)

---

## 16. RST 7.5

- Maskable interrupt.
- Highest priority among maskable interrupts.

---

## 17. RST 6.5

- Maskable interrupt.
- Medium priority.

---

## 18. RST 5.5

- Maskable interrupt.
- Lowest priority among RST interrupts.

---

## 19. INTR

- General-purpose maskable interrupt request.

---

## 20. INTA̅

- Interrupt Acknowledge signal.
- Indicates that the interrupt request has been accepted.

---

## 21. SID (Serial Input Data)

- Receives serial data one bit at a time.

---

## 22. SOD (Serial Output Data)

- Sends serial data one bit at a time.

---

## 23. VCC

- +5V power supply.

---

## 24. VSS

- Ground (0V).

---

## Summary

| Pin Group | Function |
|------------|----------|
| Address Bus | Sends higher-order address |
| Address/Data Bus | Carries lower-order address and data |
| Control Signals | Controls read, write, and memory/I/O operations |
| Interrupt Signals | Handles external interrupt requests |
| DMA Signals | Supports Direct Memory Access (DMA) |
| Clock Signals | Generates system timing |
| Reset Signals | Resets the microprocessor |
| Serial I/O | Transfers serial data |
| Power Supply | Provides power to the processor |

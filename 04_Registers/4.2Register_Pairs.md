# Register Pairs in 8085

## What is a Register Pair?

Two 8-bit registers can be combined to form a 16-bit register pair.

This allows the 8085 to store larger values or memory addresses.

---

## Register Pairs

| Register Pair | Registers |
|--------------|-----------|
| BC | B and C |
| DE | D and E |
| HL | H and L |

---

## BC Register Pair

- B stores the higher 8 bits.
- C stores the lower 8 bits.

Example:

```
B = 20H
C = 15H

BC = 2015H
```

---

## DE Register Pair

Used for storing 16-bit data.

Example:

```
D = 30H
E = 10H

DE = 3010H
```

---

## HL Register Pair

The HL pair is the most frequently used register pair.

It often stores the memory address of data.

Example:

```
LXI H, 2500H
```

Now,

H = 25H

L = 00H

HL = 2500H

---

## Summary

| Register Pair | Main Use |
|--------------|----------|
| BC | Stores 16-bit data |
| DE | Stores 16-bit data |
| HL | Memory addressing and data storage |

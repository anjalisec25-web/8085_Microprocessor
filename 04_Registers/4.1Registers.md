# Registers in 8085

## What is a Register?

A register is a small, high-speed storage location inside the microprocessor used to temporarily store data, instructions, or memory addresses during program execution.

Registers are much faster than RAM and help the CPU perform operations quickly.

---

## Registers in 8085

The 8085 microprocessor contains six general-purpose registers and one accumulator.

- A (Accumulator)
- B
- C
- D
- E
- H
- L

Each register is 8 bits wide.

---

## Accumulator (A)

The accumulator is the most important register in the 8085.

It is used for:

- Arithmetic operations
- Logical operations
- Storing intermediate results

Example:

```
MVI A, 25H
```

This loads the value 25H into the accumulator.

---

## General Purpose Registers

The registers B, C, D, E, H, and L are used to store temporary data.

Example:

```
MVI B, 10H
```

This loads the value 10H into register B.

---

## Summary

| Register | Purpose |
|----------|----------|
| A | Accumulator for arithmetic and logic operations |
| B,C,D,E,H,L | General-purpose register |

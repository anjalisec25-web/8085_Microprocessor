# Flag Register in 8085

## What is the Flag Register?

The flag register is an 8-bit register that stores the status of the result after arithmetic or logical operations.

It helps the microprocessor make decisions during program execution.

---

## Flags in 8085

There are five active flags.

| Flag | Meaning |
|------|---------|
| S | Sign Flag |
| Z | Zero Flag |
| AC | Auxiliary Carry Flag |
| P | Parity Flag |
| CY | Carry Flag |

---

## Sign Flag (S)

- Set to 1 if the result is negative.
- Otherwise, it is 0.

---

## Zero Flag (Z)

- Set to 1 if the result is zero.
- Otherwise, it is 0.

Example:

```
25H - 25H = 00H
```

Zero Flag = 1

---

## Auxiliary Carry Flag (AC)

Set when there is a carry from bit 3 to bit 4.

Used mainly in BCD arithmetic.

---

## Parity Flag (P)

- Set to 1 if the result contains an even number of 1s.
- Set to 0 if the result contains an odd number of 1s.

Example:

```
Result = 00000011
```

Two 1s → Even parity

P = 1

---

## Carry Flag (CY)

Set to 1 if an arithmetic operation generates a carry or borrow.

Example:

```
FFH + 01H
```

Result = 00H

Carry = 1

---

## Summary

| Flag | Set When |
|------|----------|
| S | Result is negative |
| Z | Result is zero |
| AC | Carry from bit 3 to bit 4 |
| P | Even parity |
| CY | Carry or borrow occurs |

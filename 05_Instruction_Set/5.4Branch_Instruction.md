# Branch Instructions in 8085

## What are Branch Instructions?

Branch instructions change the normal sequence of program execution. They are used to implement decision-making, loops, and subroutines.

---

## Common Branch Instructions

| Instruction | Description | Example |
|-------------|-------------|---------|
| JMP | Unconditional jump | JMP 2500H |
| JC | Jump if carry | JC 2500H |
| JNC | Jump if no carry | JNC 2500H |
| JZ | Jump if zero | JZ 2500H |
| JNZ | Jump if not zero | JNZ 2500H |
| JP | Jump if positive | JP 2500H |
| JM | Jump if minus | JM 2500H |
| JPE | Jump if parity even | JPE 2500H |
| JPO | Jump if parity odd | JPO 2500H |
| CALL | Call a subroutine | CALL 3000H |
| RET | Return from subroutine | RET |
| RST | Restart (software interrupt) | RST 1 |

---

## Example

```assembly
MVI A, 00H
CMP A
JZ NEXT

NEXT:
HLT
```

### Explanation

- Load 00H into A.
- Compare A with itself.
- Zero flag becomes 1.
- `JZ` transfers control to `NEXT`.
- Program stops with `HLT`.

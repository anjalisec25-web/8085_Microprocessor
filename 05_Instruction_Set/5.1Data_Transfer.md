# Data Transfer Instructions in 8085

## Instruction Format

8085 instructions are classified based on the number of bytes they occupy in memory.

| Instruction Type | Description | Example |
|------------------|-------------|---------|
| **1-Byte Instruction** | Contains only the opcode. | `MOV A, B`, `XCHG`, `HLT` |
| **2-Byte Instruction** | Contains the opcode followed by 8-bit data. | `MVI A, 25H` |
| **3-Byte Instruction** | Contains the opcode followed by a 16-bit address or data. | `LXI H, 2500H`, `LDA 2500H` |

---

## What are Data Transfer Instructions?

Data transfer instructions are used to copy data from one location to another. The source data remains unchanged after the transfer.

**These instructions do not affect any flags.**

---

## Common Data Transfer Instructions

| Instruction | Description | Example |
|-------------|-------------|---------|
| **MOV** | Copy data from one register to another. | `MOV A, B` |
| **MVI** | Load immediate data into a register. | `MVI C, 25H` |
| **LDA** | Load accumulator from a memory location. | `LDA 2500H` |
| **STA** | Store accumulator into a memory location. | `STA 2501H` |
| **LDAX** | Load accumulator indirectly from memory. | `LDAX B` |
| **STAX** | Store accumulator indirectly into memory. | `STAX D` |
| **LHLD** | Load H and L registers from two consecutive memory locations. | `LHLD 2500H` |
| **SHLD** | Store H and L registers into two consecutive memory locations. | `SHLD 2500H` |
| **LXI** | Load 16-bit data into a register pair. | `LXI H, 2500H` |
| **XCHG** | Exchange the contents of HL and DE register pairs. | `XCHG` |

---

## Example

```assembly
MVI A, 25H
MOV B, A
STA 2500H
HLT
```

### Explanation

- `MVI A, 25H` → Load `25H` into register **A**.
- `MOV B, A` → Copy the contents of register **A** into register **B**.
- `STA 2500H` → Store the contents of register **A** into memory location **2500H**.
- `HLT` → Stop program execution.

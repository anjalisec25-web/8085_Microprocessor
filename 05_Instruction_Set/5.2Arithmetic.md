# Arithmetic Instructions in 8085

## What are Arithmetic Instructions?

Arithmetic instructions perform mathematical operations such as addition, subtraction, increment, and decrement.

These instructions usually affect the flag register.

---

## Common Arithmetic Instructions

| Instruction | Description | Example |
|-------------|-------------|---------|
| ADD | Add register to accumulator | ADD B |
| ADC | Add with carry | ADC C |
| ADI | Add immediate data | ADI 20H |
| SUB | Subtract register from accumulator | SUB D |
| SBB | Subtract with borrow | SBB E |
| SUI | Subtract immediate data | SUI 10H |
| INR | Increment register by 1 | INR B |
| DCR | Decrement register by 1 | DCR C |
| INX | Increment register pair | INX H |
| DCX | Decrement register pair | DCX D |
| DAD | Add register pair to HL | DAD B |

---

## Example

```assembly
MVI A, 15H
MVI B, 10H
ADD B
HLT
```

### Explanation

- Load 15H into A.
- Load 10H into B.
- Add B to A.
- Result = 25H.

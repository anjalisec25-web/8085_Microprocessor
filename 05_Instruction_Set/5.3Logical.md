# Logical Instructions in 8085

## What are Logical Instructions?

Logical instructions perform operations on bits. They are used for comparison, masking, setting, or complementing data.

---

## Common Logical Instructions

| Instruction | Description | Example |
|-------------|-------------|---------|
| ANA | AND operation | ANA B |
| ANI | AND immediate | ANI 0FH |
| ORA | OR operation | ORA C |
| ORI | OR immediate | ORI 80H |
| XRA | XOR operation | XRA D |
| XRI | XOR immediate | XRI FFH |
| CMP | Compare with accumulator | CMP B |
| CMA | Complement accumulator | CMA |
| CMC | Complement carry flag | CMC |
| STC | Set carry flag | STC |
| RLC | Rotate left | RLC |
| RRC | Rotate right | RRC |
| RAL | Rotate left through carry | RAL |
| RAR | Rotate right through carry | RAR |

---

## Example

```assembly
MVI A, 0FH
MVI B, F0H
ORA B
HLT
```

### Explanation

- Load 0FH into A.
- Load F0H into B.
- Perform OR operation.
- Result = FFH.

# Components and Their Functions

## 1. Arithmetic Logic Unit (ALU)

- Performs arithmetic operations such as addition and subtraction.
- Performs logical operations such as AND, OR, XOR, and Compare.

---

## 2. Accumulator

- An 8-bit register used to store data.
- Works with the ALU during arithmetic and logical operations.

---

## 3. General Purpose Registers

- Six 8-bit registers: B, C, D, E, H, and L.
- Used to temporarily store data.
- Can be combined as BC, DE, and HL register pairs.

---

## 4. Program Counter (PC)

- A 16-bit register.
- Stores the address of the next instruction to be executed.

---

## 5. Stack Pointer (SP)

- A 16-bit register.
- Points to the top of the stack in memory.

---

## 6. Instruction Register and Decoder

- Stores the current instruction.
- Decodes the instruction before execution.

---

## 7. Timing and Control Unit

- Controls all operations of the microprocessor.
- Generates timing and control signals.

---

## 8. Address Buffer

- Carries the higher-order address (A8–A15).
- Unidirectional.

---

## 9. Address/Data Buffer

- Carries the lower-order address (AD0–AD7).
- Carries data during data transfer.
- Bidirectional and multiplexed.

---

## 10. Interrupt Control

- Handles interrupt requests.
- Supports TRAP, RST 7.5, RST 6.5, RST 5.5, and INTR.

---

## 11. Serial Input/Output Control

- Supports serial communication.
- Uses SID (Serial Input Data) and SOD (Serial Output Data).

---

## Summary

| Component | Function |
|-----------|----------|
| ALU | Performs arithmetic and logical operations |
| Accumulator | Stores data for ALU operations |
| General Purpose Registers | Temporarily stores data |
| Program Counter | Holds the address of the next instruction |
| Stack Pointer | Points to the top of the stack |
| Instruction Register | Stores and decodes instructions |
| Timing & Control Unit | Controls processor operations |
| Address Buffer | Carries higher-order address |
| Address/Data Buffer | Carries lower-order address and data |
| Interrupt Control | Handles interrupts |
| Serial I/O Control | Manages serial communication |

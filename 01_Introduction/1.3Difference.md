# Difference Between Microprocessor and Microcontroller

| Microprocessor | Microcontroller |
|----------------|-----------------|
| Contains only the CPU. | Contains CPU, memory, and I/O ports on a single chip. |
| Requires external memory and peripherals. | Has built-in memory and peripherals. |
| Used for general-purpose computing. | Used for specific or dedicated tasks. |
| Higher power consumption. | Lower power consumption. |
| Higher processing speed. | Moderate processing speed. |
| More expensive due to external components. | More cost-effective. |
| Larger system size. | Compact system size. |
| Examples: Intel 8085, Intel 8086 | Examples: 8051, PIC, AVR |

# Features of 8085

- 8-bit microprocessor.
- 16-bit address bus.
- 8-bit data bus.
- Can address up to **64 KB** of memory.
- Operates on a **+5V** power supply.
- Available in a **40-pin** DIP package.
- Supports **5 hardware interrupts**.
- Has **serial input and serial output** ports.
- Supports both **hardware and software control**.
- Provides a built-in clock generator.

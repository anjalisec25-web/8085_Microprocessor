# Stack in 8085

## What is a Stack?

A **Stack** is a temporary memory area used to store data during program execution. It follows the **LIFO (Last In, First Out)** principle.

---

## Stack Pointer (SP)

The **Stack Pointer (SP)** is a 16-bit register that stores the address of the top of the stack.

- The stack grows towards lower memory addresses.
- The Stack Pointer is automatically updated during stack operations.

---

## Stack Operations

### PUSH Operation

- Stores data onto the top of the stack.
- The Stack Pointer is decremented before storing the data.

---

### POP Operation

- Retrieves data from the top of the stack.
- The Stack Pointer is incremented after retrieving the data.

---

## Uses of Stack

- Store temporary data.
- Save register contents.
- Store return addresses during subroutine calls.
- Handle interrupts.

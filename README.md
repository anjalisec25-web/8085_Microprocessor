# 8085 Microprocessor

A beginner-friendly repository for learning the fundamentals of the "8085 Microprocessor". This repository contains well-organized notes, diagrams, instruction sets, registers, and assembly language programs. It is intended for students, especially those studying Electronics and Communication Engineering (ECE), Computer Science, and related fields.

## Repository Structure

8085-Microprocessor/
│
├── 01_Introduction
├── 02_Pin_Diagram
├── 03_Architecture
├── 04_Registers
├── 05_Instruction_Set
├── 06_Programs
└── README.md
```

## Topics Covered

- Introduction to Microprocessors
- Microprocessor vs Microcontroller
- Introduction to the 8085 Microprocessor
- Features of the 8085
- Pin Diagram and Pin Functions
- 8085 Architecture
- Registers and Flag Register
- Instruction Set
- Assembly Language Programming
- Basic 8085 Programs

## Purpose

This repository is created to:
- Learn the fundamentals of the 8085 Microprocessor.
- Organize study notes in a structured manner.
- Practice 8085 Assembly Language Programming.
- Serve as a quick revision guide.

## Learning Outcome

After completing this repository, you will be able to:
- Understand the architecture of the 8085 Microprocessor.
- Explain the functions of its internal components.
- Identify and use different instruction types.
- Write and understand basic 8085 Assembly Language programs.
